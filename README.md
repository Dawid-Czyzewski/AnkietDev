## Opis

AnkietDev to lekka i intuicyjna platforma umożliwiająca tworzenie ankiet online, zbieranie odpowiedzi i przeglądanie statystyk. Rejestracja jest szybka (tylko e‑mail, hasło, potwierdzenie hasła), a po zalogowaniu użytkownik może od razu utworzyć swoją pierwszą ankietę.

## Funkcje

* Prosta rejestracja (e‑mail, hasło, potwierdzenie hasła) — brak konieczności potwierdzania konta
* Tworzenie ankiet z pytaniami otwartymi i zamkniętymi
* Dowolna liczba opcji odpowiedzi w pytaniach zamkniętych
* Edycja, usuwanie i podgląd ankiet z poziomu dashboardu
* Widok statystyk: wartości procentowe, liczba odpowiedzi per pytanie, podgląd odpowiedzi otwartych
* Strona kontaktowa z formularzem zgłoszeń i sugestii funkcji

## Technologie

* Backend: PHP (czyste PHP, bez frameworków)
* Baza danych: MariaDB
* Frontend: React + Tailwind CSS

## Description

AnkietDev is a lightweight, intuitive platform for creating online surveys, collecting responses, and viewing clear statistics. Registration is quick (email, password, confirm password), and users can start creating surveys immediately after logging in.

## Features

* Simple registration (email, password, confirm password) — no email confirmation required
* Create surveys with open and closed questions
* Unlimited answer options for closed questions
* Edit, delete and preview surveys from the dashboard
* Results view: percentage breakdowns, answer counts per question, preview of open responses
* Contact page with form for help requests and feature suggestions

## Technologies

* Backend: PHP (pure PHP, no frameworks)
* Database: MariaDB
* Frontend: React + Tailwind CSS
